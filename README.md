## *Installation*

1. Clone the repository:
   bash
   git clone https://github.com/Hersheys-commits/E-Cell-Task.git

2. Navigate to the project directory:
   bash
   cd E-Cell-Task

3. Install dependencies:
   bash
   npm install

4. Run the app locally:
   bash
   npm start
